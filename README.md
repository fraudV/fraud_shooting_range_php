## 系统安装

源码下载
~~~
https://github.com/telllieV/fraud_shooting_range_php.git
~~~

进入目录
~~~
cd fraud_shooting_range_php
~~~
下载PHP依赖包（文件所在根目录执行如下命令）
~~~
composer install
~~~
安装docker，环境使用是**docker compose** 命令启动，不支持**docker-compose**
~~~
curl -s https://get.docker.com/ | sh 
~~~
